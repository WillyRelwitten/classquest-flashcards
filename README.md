# 🏫 ClassQuest - 3D Flash Cards for Super Students!

A fun, mobile-first classroom flashcard game for kids (grades 1-12). Built with Three.js 3D cards, pure HTML, CSS, and JavaScript.

## ✨ New in This Update
- **Three.js 3D Cards** — Tap the beautiful 3D card to flip it with smooth animation
- **10 Questions per Grade** — Every grade now has 10+ carefully selected questions
- **Improved Visuals** — Nicer textures, lighting, and borders on cards

## Features
- **Grade Selection**: Pick any grade from 1 to 12
- **3D Flip Cards**: Tap the Three.js card to reveal answers with realistic 3D rotation
- **Multiple Choice**: 4 options per question
- **Scoring & Progress**: Track your score live
- **Celebrations**: Fireworks and confetti on perfect scores!
- **Surprise Me**: Random grade button

## How to Play
1. Open `index.html` in any modern browser (Chrome, Firefox, Safari, Edge)
2. Choose your grade level
3. Tap the 3D card to flip and see the answer
4. Select your answer from the 4 choices
5. Aim for 100% to trigger the celebration!

## Tech Stack
- Three.js r128 (via CDN)
- Tailwind CSS (via CDN)
- Canvas Confetti
- Vanilla JavaScript
