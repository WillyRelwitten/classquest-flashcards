# 🏫 ClassQuest - Flash Cards for Super Students!

A fun, mobile-first classroom flashcard game for kids (grades 1-12). Built with pure HTML, CSS, and JavaScript.

## Features
- **Grade Selection**: Pick any grade from 1 to 12
- **Flip Cards**: Tap to reveal answers
- **Multiple Choice**: 4 options per question
- **Scoring & Progress**: Track your score as you go
- **Celebrations**: Fireworks and confetti on perfect scores!
- **Surprise Me**: Random grade button for fun

## How to Play
1. Open `index.html` in any browser
2. Choose your grade level
3. Answer the questions by tapping the cards and selecting answers
4. Aim for 100% to trigger the celebration!

## Tech Stack
- Tailwind CSS (via CDN)
- Canvas Confetti for celebrations
- Vanilla JavaScript
